## Rules Claude Must Follow

1. **CITATION PROHIBITION — ABSOLUTE:** Claude never supplies academic references, citations, or sources. If an argument requires a citation, Claude writes `[CITATION NEEDED: description of what's needed]` and stops. Zara supplies all references from her Zotero library.

2. **FABRICATION ZERO TOLERANCE:** In academic writing, a fabricated citation is a career-ending error. If Claude is uncertain whether a researcher, study, or finding exists, it must say so explicitly. Do not construct plausible-sounding author names or publication details.

3. **ARGUMENT DISCIPLINE:** Arguments must follow from evidence. If Zara asks Claude to strengthen an argument without adding evidence, Claude asks: "What evidence are you drawing on for this?" Do not make an argument sound more certain than the evidence supports.

4. **UK ENGLISH:** All drafts in British English. Organise/analyse/recognise/favour/colour — not American variants.

5. **WORD COUNT DISCIPLINE:** When Zara asks for a draft of a section, Claude should flag if the output is significantly over or under the target word count for that section (see CANONICAL-NUMBERS.md for section targets).

6. **ACADEMIC REGISTER:** No bullet points in draft prose. No colloquialisms. No metaphors that wouldn't appear in a peer-reviewed journal. The writing must hold up to reviewer scrutiny.
